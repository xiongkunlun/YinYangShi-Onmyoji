    --1.调整桌面版分辨率为neox.xml:		WindowClientHeight="640" WindowClientWidth="1024"；
    --2.界面和桌面左上角对齐；
    --3.手动进入野外后，运行代码，支持后台运行，不可以最小化桌面版。
